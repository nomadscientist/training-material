# Profiling of microbial communities using targeted and shotgun metagenomics

This five-day training course, organised by **ELIXIR-IT**, provides a comprehensive overview of modern microbiota analysis techniques — from sample preparation and bioinformatics analysis to machine learning applications.

The course is designed for biological and biomedical PhD students and Post-Docs who want to develop practical skills in analysing microbial communities using both **targeted** (16S/ITS metabarcoding) and **untargeted** (shotgun metagenomics) approaches.

## Topics

- DNA metabarcoding (16S/ITS amplicon sequencing) with QIIME2 and DADA2
- Shotgun metagenomics: taxonomic and functional profiling with MetaPhlAn, Kraken2/Bracken and HUMAnN
- Strain-level profiling with StrainPhlAn
- Metagenome-assembled genomes (MAGs) with Metabat2 and kMetaShot
- Machine learning approaches for microbiome data analysis
- Inference of microbial association networks (NetCoMi, SpiecEasi)
- Galaxy platform and FAIRyMAGs workflow

## Learning Objectives

- Understand the principles of DNA metabarcoding and shotgun metagenomics
- Perform quality control and preprocessing of raw sequencing data
- Analyse microbial community diversity using ecological measures (alpha and beta diversity)
- Apply multivariate statistics and dimensionality reduction to microbiome data
- Use machine learning for microbiota data interpretation and explainability
- Run bioinformatics workflows on the Galaxy platform

## Prerequisites

Participants are expected to have:

- Basic understanding of the UNIX shell (bash)
- An active Google Account
