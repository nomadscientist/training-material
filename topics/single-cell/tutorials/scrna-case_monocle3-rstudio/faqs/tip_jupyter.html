This is the most recommended way to complete this tutorial (unless you have monocle3 already installed) as it is the most reliable and visually appealing.

Hands-on: Downloading the tutorial notebook
  1. You will need to download the tutorial notebook locally to your own computer. Simply go to the Supporting Materials section in the Overview box at the very beginning of this tutorial and click on Jupyter Notebook. It will be downloaded to your computer as single-cell-scrna-case_monocle3-rstudio.ipynb file.
Hands-on: Launch JupyterLab

Currently JupyterLab in Galaxy is available on Live.useGalaxy.eu, usegalaxy.org and usegalaxy.eu.

Hands-on: Run JupyterLab
  1. Interactive Jupyter Notebook Tool: interactive_tool_jupyter_notebook :
  2. Click Run Tool
  3. The tool will start running and will stay running permanently
  4. Click on the User menu at the top and go to Active Interactive Tools and locate the JupyterLab instance you started.
  5. Click on your JupyterLab instance

If JupyterLab is not available on the Galaxy instance:

  1. Start Try JupyterLab
